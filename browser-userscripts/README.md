# Browser User-Scripts

A collection of browser user-scripts.

For using these user-scripts, a browser add-on like [Violentmonkey](https://github.com/violentmonkey/violentmonkey) is required to be installed.

<br />

## TheWorker-Bypass
Remove blur and mid screen banner, and enable right-click and selection on [theworker.co.il](https://theworker.co.il).  
[Install Link](https://raw.githubusercontent.com/MichaelYochpaz/User-Scripts/main/browser-userscripts/TheWorker-Bypass.user.js)

<br />

## RealPython-Bypass
Remove registration / login pop-up on [realpython.com](https://realpython.com).  
[Install Link](https://raw.githubusercontent.com/MichaelYochpaz/User-Scripts/main/browser-userscripts/RealPython-Bypass.user.js)
